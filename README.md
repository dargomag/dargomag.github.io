# Архив сайта Путь Дарго

Статическая архивная копия основного сайта, созданная для долгосрочного сохранения контента.

## Основная информация 
- **Оригинальный сайт:** [http://dargo.ucoz.ru/] (http://dargo.ucoz.ru/)
- **Обновляется:** [вручную]

## Структура
- `/articles` — статьи и публикации
- `/images` — изображения с сайта
- `/backups` — полные резервные копии

## Цель
Обеспечение цифрового сохранения контента на случай недоступности основной платформы.
